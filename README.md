# Netkit
Netkit é uma tool de canivete suiço. 
Com ele você pode:

• Escutar portas;

• Se conectar a hosts;

• Após se conectar ou começar a escutar uma porta pode executar algo;

• Usar como dns resolver;

• Usar como reverse dns resolver.
